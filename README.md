# Vannusic

## 功能清单
- 仿网易云音乐风格，旋转黑胶碟片播放器
- 播放本地音乐，以文字形式分享音乐
- 获取百度音乐榜单，播放/下载在线音乐，以链接形式分享在线音乐
- 顺序/随机/重复播放
- 在媒体中心显示服务
- 拔出耳机暂停音乐

## 食用方法
- git clone https://github.com/survivingME/Vannusic.git
- 打开AS编译运行~
